# gfbson

A Rust crate for parsing Good-Feel's BSON files.

## Features

- [x] Binary Files
    - [x] Reading
        - [x] Big Endian
        - [x] Little Endian
    - [x] Writing
        - [x] Big Endian
        - [x] Little Endian
- [x] JSON Files
    - [x] Reading
    - [x] Writing

To access JSON parsing, use the `json` feature when adding this crate to your project.
